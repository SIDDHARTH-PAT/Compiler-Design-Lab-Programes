# Compiler-Design-Lab-Programes


This GIT repository contains a list of workshops and labs for compilers' design
lecture. 

## Agenda:

* String Validator.
* Multi line and single line .
* Whitespace remover.
* First().
* Follow().
* Left recursion.
* Left factoring.
* Recursive Descent Parser.
* LL(1) Parser.
* Operator Precedence parser.
* Three Address Code.
